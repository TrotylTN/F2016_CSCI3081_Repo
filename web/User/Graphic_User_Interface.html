Graphic User Interface(GUI)

In GUI, processing a stack of images in one shot is not available but it provides the preview feature.

The following steps show how to use MIA in GUI:

  1. This is the first view of the program.

  2. Click into the directory the images saved, in this case, it is resources/test-images/brainstack.

  3. Double click onto the image you want to load and click “Load Canvas” butto

  4. Type in the blurriness you want in the textbox pointed in the screenshot. Then click “Apply” button to apply the blur filter.

  5. Type in the number of segments you want in the textbox pointed in the screenshot. Then click “Apply” button.

  6. To save this current version of the image, click into the directory you want to save your image and type the name of the image inside the textbox pointed. Then click “Save Canvas” button.

  7. To view other images, repeat step 2 - 3.

  8. To view the next or previous image in number sequence, click “Next Image” button or “Previous Image” button.
