Converting and Segmenting a CT Scan Using Mia

Command Line Interface:

For specific CT Scan image, the following commands could help:

  1. Make a new directory, for example, “target”. 
    • mkdir -p target
  2. Make slightly blur to reduce noise. 
    • ./build/bin/MIA resources/brainstack/mrbrain-8bit001.png -blur 1.0 target/mrbrain-8bit001.png
  3. Quantize the result into 4 segments. 
    • ./build/bin/MIA target/mrbrain-8bit001.png -quantize 4 target/mrbrain-8bit001.png

      • Or process the image in one command:

    • ./build/bin/MIA target/mrbrain-8bit001.png -quantize 4 target/mrbrain-8bit001.png

To convert a sequenced CT Scan image stack:

  1. Make a new directory, for example, “target”. 
    • mkdir -p target
  2. Make slight blur and quantize them into 4 segments. (# means 0-9) 
    • ./build/bin/MIA resources/brainstack/mrbrain-8bit0##.png -blur 1.0 -quantize 4 target/mrbrain-8bit0##.png
