Introduction

FlashPhoto

Flashphoto is a program that allows users to draw anything they want and load in photos for editing. Flashphoto provides 2 interfaces, Graphic User Interface(GUI) and Command Line Interface. The GUI provides all features for the user while the Command Line Interface only provides the user to edit their photos with filters only. For the GUI, we used GLUI as the base support of the program user interface. Flashphoto only supports PNG and JPEG images.

Medical Imaging Assistant (a.k.a. MIA)

MIA is a very similar program compared to Flashphoto but it is designed specially for medical use. MIA did not provides as many features compared to Flashphoto but it has most of the filters, a red pen tool and a marker tool for users. MIA, same as Flashphoto, also provides GUI and Command Line Interface. MIA also only supports PNG and JPEG images.
